# 11 部署运行说明

## 1. 本地环境要求

| 软件 | 版本建议 |
|---|---|
| JDK | 17 或 21 |
| Node.js | 20+ |
| MySQL | 8.x |
| Redis | 7.x |
| Maven | 3.9+ |
| Docker | 可选 |

## 2. 后端启动

### 2.1 修改配置

文件：

```text
backend/src/main/resources/application.yml
```

示例：

```yaml
spring:
  datasource:
    url: jdbc:mysql://localhost:3306/driving_school_schedule?useUnicode=true&characterEncoding=utf8&serverTimezone=Asia/Shanghai
    username: root
    password: your_password
  data:
    redis:
      host: localhost
      port: 6379
      password:
```

### 2.2 初始化数据库

执行：

```text
docs/sql/init.sql
```

或将 `05_数据库设计.md` 中的 SQL 整理为 init.sql 执行。

### 2.3 启动后端

```bash
cd backend
mvn spring-boot:run
```

默认端口：

```text
8080
```

## 3. 前端启动

```bash
cd frontend
npm install
npm run dev
```

默认端口：

```text
5173
```

## 4. Docker Compose 建议

可创建：

```text
docker-compose.yml
```

包含：

- MySQL；
- Redis；
- 后端服务；
- 前端 Nginx 服务。

示例：

```yaml
services:
  mysql:
    image: mysql:8.0
    environment:
      MYSQL_ROOT_PASSWORD: root
      MYSQL_DATABASE: driving_school_schedule
    ports:
      - "3306:3306"

  redis:
    image: redis:7
    ports:
      - "6379:6379"

  backend:
    build: ./backend
    ports:
      - "8080:8080"
    depends_on:
      - mysql
      - redis

  frontend:
    build: ./frontend
    ports:
      - "80:80"
    depends_on:
      - backend
```

## 6. 演示数据建议

- 教练 5 名；
- 车辆 6 辆；
- 学员 40 名；
- 每个教练默认半天最多 5 人；
- 明天上午设置一个教练满员；
- 另一个教练空闲，方便演示调剂。

## 7. 常见问题

### 7.1 预约接口报满员

检查：

- 排班 current_students 是否已经等于 max_students；
- 排班 status 是否为 OPEN；
- 是否重复预约；
- Redis 锁是否未释放。

### 7.2 前端跨域

后端配置 CORS，允许前端开发地址：

```text
http://localhost:5173
```

### 7.3 Redis 连接失败

检查：

- Redis 是否启动；
- host/port 是否正确；
- 是否设置密码。

## 8. 打包部署

前端：

```bash
npm run build
```

后端：

```bash
mvn clean package
java -jar target/driving-school.jar
```
