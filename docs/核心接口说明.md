# 核心接口说明

## 基础信息

- 基础路径: `/api`
- 统一响应: `{ code, message, data }`
- 认证方式: `Authorization: Bearer <JWT Token>`

## 认证模块 (auth)

| 方法 | 路径 | 权限 | 说明 |
|------|------|------|------|
| POST | /auth/login | 无 | 登录，返回 JWT Token + userId + role + relatedId |
| GET | /auth/me | 登录 | 获取当前用户信息 |

## 账号管理 (user)

| 方法 | 路径 | 权限 | 说明 |
|------|------|------|------|
| GET | /users | ADMIN | 分页查询账号列表 |
| POST | /users | ADMIN | 创建教练/学员账号 |
| PATCH | /users/{id}/status | ADMIN | 启用/停用账号 |
| PATCH | /users/{id}/password/reset | ADMIN | 重置密码 |

## 教练模块 (coach)

| 方法 | 路径 | 权限 | 说明 |
|------|------|------|------|
| GET | /coaches | ALL | 分页查询（教练只看自己） |

| POST | /coaches | ADMIN | 新增教练 |
| PUT | /coaches/{id} | ADMIN | 修改教练 |
| PATCH | /coaches/{id}/status | ADMIN | 状态修改 |
